Enterprise Pro Prototype

Description:
This prototype processes seizure data using Python.

Features:
- Import CSV / Excel data
- Filter data by date, category, and location
- Keyword search across dataset
- Top 10 summary generation
- Geocode postcode / city
- Export results to CSV and PDF

How to run:
1. Install Python packages:
   pip install pandas requests reportlab openpyxl

2. Run the backend:
   python Backend.py

Outputs:
- filtered_output.csv
- geo_output.csv
- report.pdf
